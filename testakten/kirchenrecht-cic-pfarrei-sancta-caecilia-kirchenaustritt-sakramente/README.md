## Arbeitsauftrag

Die Akte prüft, ob das Plugin Norm, Katechismus, pastoralen Ton und Mehrsprachigkeit zugleich beherrscht. Besonders wichtig: staatlicher Kirchenaustritt ist nicht Tauflöschung; Taufe bleibt sakramental unauslöschlich; dennoch kann eine öffentliche Distanzierung innerkirchliche Folgen haben.
