# Testakte JVEG Zeugin Berger / LG Tübingen

Diese Beispielakte enthält die vom Nutzer bereitgestellten Unterlagen zum Vorschuss- und Zeugenentschädigungsfall Sophia Berger. Sie ist für das Plugin `jveg-kostenpruefer` gedacht.

## Enthaltene Originalunterlagen

- `antrag zeugin berger.pdf`
- `antrag_zeugin_berger.pdf`
- `gerichtsschreiben_lg_tuebingen.pdf`
- `schreiben_lg_tuebingen.docx`
- `schriftsatz_vorschussantrag.pdf`

## Testablauf

1. `jveg-kommandocenter` starten.
2. Alle Unterlagen strippen und die Kostenpositionen extrahieren.
3. Vorschussfähigkeit nach § 3 JVEG prüfen.
4. Fahrtkosten, Übernachtung und Verdienstausfall getrennt rechnen.
5. Gerichtsschreiben vom 27.01.2026 auf Tatbestands- und Ermessensfehler prüfen.
6. Prüfbericht und Antwort-/Festsetzungsentwurf erzeugen.
