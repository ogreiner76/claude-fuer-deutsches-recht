# 14 — Fussnoten-Format: Langzitate, Kurzzitate, Wiederholungszitate

**Kanzleihandbuch Zitierweise v4.0 — Fussnoten-Standard**
**Stand: Oktober 2025**

---

## A. Langzitat (Erstnennung / Vollbeleg)

Das Langzitat wird bei der ersten Erwähnung einer Quelle verwendet. Es enthalt alle Pflichtangaben.

### A.1 Rechtsprechung — Langzitat
```
BGH, Urt. v. 21.12.2021, Az. VI ZR 48/21, BGHZ 232, 133 (Rn. 18), NJW 2022,
1238 (1240), abrufbar unter: https://dejure.org/2021,40001 (abgerufen am 01.10.2025).
```

### A.2 Literatur Kommentar — Langzitat
```
Gruenenberg/Ellenberger, BGB, 83. Aufl. 2024, § 242 Rn. 27.
```

### A.3 Literatur Aufsatz — Langzitat
```
Rolfs, Massenentlassung und europarechtliche Vorgaben, NZA 2023, 713 (715).
```

---

## B. Kurzzitat (Wiederholung, nicht unmittelbar folgend)

Das Kurzzitat wird bei erneuter Erwähnung einer Quelle verwendet, wenn zwischen Erst- und Wiederholungsnennung andere Quellen zitiert werden.

### B.1 Rechtsprechung — Kurzzitat
```
BGHZ 232, 133 (Rn. 18).
BGH NJW 2022, 1238 (1240).
BVerfGE 7, 198 (205).
```

### B.2 Literatur Kommentar — Kurzzitat
```
Gruenenberg/Ellenberger (Fn. X), § 242 Rn. 27.
```
oder
```
Gruenenberg/Ellenberger, a.a.O. (Fn. X), Rn. 27.
```

### B.3 Literatur Aufsatz — Kurzzitat
```
Rolfs (Fn. X), NZA 2023, 713 (716).
```

---

## C. Wiederholungszitat (unmittelbar folgend)

Wenn dieselbe Quelle in der unmittelbar folgenden Fussnote oder auf derselben Seite erneut zitiert wird:

```
A.a.O., Rn. 28.
```
oder
```
Ebenda, S. 1241.
```

**Kanzleistandard:** a.a.O. bevorzugt. Ebenda nur in historischen oder fremdsprachigen Texten.

---

## D. Fussnoten-Nummer und -Format

- Hochgestellte Zahl im Text: `¹` (Fliesstext) → Fussnote am Seitenende
- Fussnoten-Schrift: 2 Punkt kleiner als Fliesstext
- Punkt am Ende der Fussnote: immer
- Fussnoten-Trennlinie: dünne Linie (kanzleiinterne Vorlage)

---

## E. Besondere Situationen

### E.1 Zwei Quellen in einer Fussnote
```
BGH NJW 2022, 1238 (1240); Gruenenberg/Ellenberger, BGB, 83. Aufl. 2024, § 242 Rn. 27.
```
Quellen werden durch Semikolon getrennt.

### E.2 Ablehnende Literatur
```
A.A. Palandt/Ellenberger, BGB, 80. Aufl. 2021, § 242 Rn. 28.
```

### E.3 Weitere Nachweise (m.w.N.)
```
BGH NJW 2022, 1238 (1240) m.w.N.
```

### E.4 Mit ausführlicher Begr.
```
Rolfs, NZA 2023, 713 (715 ff. — ausfuehrlich).
```
