# Testakte Vertragsausfüller - BSAG Kiosk Huckelriede

Diese Beispielakte enthält zwei Word-Dokumente für den Vertragsausfüller:

- `BSAG-Mietvertrag-Vorlage.docx`
- `BSAG-TermSheet-Kiosk-Huckelriede - Kopie.docx`

## Testablauf

1. Starte `vaf-kommandocenter`.
2. Lasse die Vorlage strippen und ein Feldinventar erzeugen.
3. Mappe das Term Sheet auf die Mietvertragsvorlage.
4. Lasse offene Rückfragen ausgeben.
5. Erstelle einen Clean-Entwurf.
6. Erst wenn ausdrücklich gewünscht: Track-Changes- oder Redline-Fassung vorbereiten.
