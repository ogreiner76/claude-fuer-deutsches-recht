# Datenschutz- und KI-Compliance Startcheck

Roeschen Tech analysiert Kundenformulare mit einem KI-Modul. Unklar:

- Werden personenbezogene Daten zum Training genutzt?
- Gibt es AV-Verträge mit Pilotkunden?
- Ist ein KI-System im Sinne der KI-VO dokumentiert?
- Gibt es Hochrisiko-Anzeichen, wenn Kunden das Tool im HR-Kontext verwenden?
- Enthält das Investorendeck übertriebene Aussagen zur Compliance?

Notiz: Für Closing sollten Datenschutzrecht und KI-VO-Checker hinzugezogen werden.
