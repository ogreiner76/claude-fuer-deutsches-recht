# Track-Changes-Nachfrage

Bevor eine Änderungsfassung erstellt wird:

**Soll zusätzlich eine Track-Changes- oder Redline-Fassung erstellt werden?**

- Ja: Ausgangsdokument und Zielentwurf werden verglichen, Änderungen werden protokolliert.
- Nein: Es bleibt bei Clean-Vertrag, Ausfüllprotokoll und offener-Punkte-Liste.
