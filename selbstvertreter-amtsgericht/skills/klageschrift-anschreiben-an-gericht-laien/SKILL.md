# Wie schreibe ich foermlich ans Amtsgericht?

## Worum geht es?

Ein Gericht ist eine Behörde mit eigenen Konventionen. Sie muessen nicht juristisch formuliert schreiben — aber bestimmte Foermlichkeiten erhoehen Ihre Glaubwuerdigkeit und vermeiden Rueckfragen. Diese Skill zeigt die richtigen Anreden, Briefkopf-Elemente und uebliche Schlussformeln.

## Wann brauchen Sie diese Skill?

- Sie schreiben einen Schriftsatz ans Amtsgericht.
- Sie sind unsicher, wie Sie das Gericht anreden.
- Sie wollen das richtige Aktenzeichen-Format treffen.

## Fachbegriffe (kurz erklaert)

- **Aktenzeichen**: Eindeutige Nummer eines Verfahrens, z. B. "5 C 234/25".
- **Schriftsatz**: Jedes schriftliche Eingabe ans Gericht.
- **Geschaeftsstelle**: Buero des Gerichts, das Schriftsaetze entgegennimmt.

## Rechtsgrundlagen

- **§ 130 ZPO** — Inhalt eines Schriftsatzes.
- **§ 130a ZPO** — Elektronisches Dokument.
- **§ 253 ZPO** — Klageschrift Form.

## Schritt-für-Schritt-Anleitung

### Schritt 1 — Briefkopf

Oben links Ihre Daten:

```
[Vorname Nachname]
[Strasse Hausnummer]
[PLZ Ort]

Telefon: [optional]
Email: [optional]
```

Bei juristischen Personen Firma + Vertreter.

### Schritt 2 — Datum und Ort

Oben rechts oder unter Ihrem Namen:

```
Musterstadt, 27. Mai 2026
```

### Schritt 3 — Empfaenger

```
An das
Amtsgericht Musterstadt
- Geschaeftsstelle der Zivilabteilung -
Musterstrasse 1
12345 Musterstadt
```

Bei Bekanntem Aktenzeichen: konkrete Abteilung angeben, z. B. "Abt. C 5".

### Schritt 4 — Betreffzeile mit Aktenzeichen

```
Betreff: Aktenzeichen [AZ]
 Rechtsstreit Mustermann ./. Beispielmann
 wegen [Streitgegenstand]
```

Bei erster Klage (noch kein Aktenzeichen):

```
Betreff: Klage
 Mustermann ./. Beispielmann
 wegen [Streitgegenstand]
```

Praxis-Hinweis: "./." (Punkt-Strich-Punkt) bedeutet "gegen".

### Schritt 5 — Anrede

```
Sehr geehrte Damen und Herren,
```

Oder bei konkretem Richter (selten benoetigt):

```
Sehr geehrter Herr Vorsitzender,
```

Im Allgemeinen: "Sehr geehrte Damen und Herren" ist immer korrekt.

### Schritt 6 — Eroeffnungssatz

Erster Schriftsatz (Klage):

```
in der Sache [Name] ./. [Name] erhebe ich
hiermit Klage und stelle folgende Antraege:
```

Weitere Schriftsaetze:

```
in obiger Sache nehme ich wie folgt Stellung:
```

oder

```
in obiger Sache repliziere ich auf die
Klageerwiderung vom [Datum] wie folgt:
```

### Schritt 7 — Inhalt

Tatsachenvortrag, Beweisangebot, Antraege — wie in den anderen Skills beschrieben.

### Schritt 8 — Schlussformel

```
Mit freundlichen Gruessen

[Unterschrift]

[Name in Druckbuchstaben]
```

Bei elektronischer Einreichung ueber MJP: einfache Signatur reicht in Verbindung mit dem sicheren Uebermittlungsweg (§ 130a III ZPO).

### Schritt 9 — Anlagen

Am Ende:

```
Anlagen:
K1 — [Bezeichnung]
K2 — [Bezeichnung]
...
```

### Schritt 10 — Empfaenger-Liste

Bei einem Versand an mehrere (Gericht + Beklagter): Verteiler unten:

```
Verteiler:
1 x Original Amtsgericht Musterstadt
1 x Abschrift Beklagter
1 x Kopie zu meinen Akten
```

## Worauf Sie besonders achten muessen

- **Sie-Form, Hoeflichkeitsform**.
- **Keine Beschimpfungen**, auch bei berechtigter Empoerung.
- **Keine Umgangssprache** ("Der hat mich verarscht" — schreiben Sie es sachlich).
- **Aktenzeichen immer angeben**, sobald Sie es haben — sonst landen Schriftsaetze evtl. nicht in der richtigen Akte.

### Schritt 11 — Anrede zum Gericht selbst

Beim Amtsgericht entscheidet i. d. R. ein Einzelrichter, **keine** Kammer (Kammer ist die Spruchkoerper-Bezeichnung am Landgericht). Verwenden Sie also am AG die Formulierungen "Das Gericht" oder "Der Einzelrichter" — nicht "die Kammer".

Beispiel:

```
Ich bitte das Gericht, den Beweisantrag zu beruecksichtigen.
```

oder

```
Ich bitte den Einzelrichter, ...
```

## Typische Fehler

- "Hallo Herr Richter, ich schreibe wegen ..." → Zu lax.
- "Sehr geehrter Herr Schmidt" — zwar persoenlich, aber wenn Sie nicht **sicher** wissen, dass Richter Schmidt zuständig ist, lieber neutral.
- "Aktenzeichen weglassen, das Gericht weiss es ja." → Pflicht-Angabe, sonst Zuordnungs-Probleme.
- "Mehrere Sachen in einem Brief vermischen." → Pro Verfahren separater Schriftsatz.
