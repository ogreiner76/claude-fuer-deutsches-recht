# Anlagen formatieren — K1, K2 oben rechts, Schrift 12pt

## Worum geht es?

Wenn Sie Beweisstuecke (Anlagen) einreichen, sollten diese **einheitlich beschriftet** sein, damit der Richter und der Gegner sie auf einen Blick finden. Diese Skill zeigt Ihnen, **wie** Sie das machen: welche Schriftart, welche Groesse, wo genau die Beschriftung hingehoert und wie Sie das in Word, LibreOffice oder einem PDF-Programm umsetzen.

## Wann brauchen Sie diese Skill?

- Sie erstellen Anlagen am Computer (PDF, Word, LibreOffice).
- Sie wollen, dass Ihre Klage professionell aussieht.
- Sie scannen Papier-Dokumente ein und wollen sie mit "Anlage K1" beschriften.

## Fachbegriffe (kurz erklaert)

- **PDF**: Ein Dokument-Format, das auf jedem Geraet gleich aussieht. Standard für Gerichts-Einreichungen.
- **OCR**: Optical Character Recognition — eine Funktion, die gescannte Bilder in durchsuchbaren Text umwandelt. Wichtig für das Gericht, weil Text durchsucht werden kann.
- **Stempel**: Eine Beschriftung, die auf das fertige PDF aufgebracht wird (auch "Stempel-Annotation" genannt).

## Empfohlene Formatierung (Konvention)

| Element | Empfehlung |
|------------------------|-----------------------------------------------------------|
| Schriftart Klageschrift| Times New Roman, Arial oder Calibri |
| Schriftgroesse Text | 12 Punkt |
| Zeilenabstand | 1,5-fach |
| Rand oben/unten | rd. 2,5 cm |
| Rand links | rd. 3 cm (Platz für Heftung) |
| Rand rechts | rd. 2 cm |
| Anlage-Beschriftung | oben rechts, Schriftgroesse 14 Punkt, fett |
| Position Anlage K1 | etwa 2 cm vom oberen Rand, etwa 2 cm vom rechten Rand |
| Seitenzahlen | unten zentriert oder unten rechts, fortlaufend |

**Wichtig:** Das ist eine **Konvention der Praxis**, kein Gesetz. Wenn Sie davon abweichen, ist die Klage nicht unwirksam. Aber: Eine saubere Formatierung wirkt sich positiv aus.

## Schritt-für-Schritt-Anleitung — Word/LibreOffice

### Schritt 1 — Klageschrift in 12pt schreiben

In Word/LibreOffice:
- Markieren Sie den ganzen Text (Strg+A).
- Stellen Sie die Schriftart ein: Times New Roman oder Arial.
- Stellen Sie die Schriftgroesse ein: 12 Punkt.
- Stellen Sie den Zeilenabstand ein: 1,5 (im Format-Menue oder ueber das Absatz-Dialogfeld).

### Schritt 2 — Anlagen als separate Dateien erstellen

Jede Anlage wird **ein eigenes PDF**. Erstellen Sie es so:

**Bei einem gescannten Dokument** (z.B. Vertrag in Papierform):
- Scannen mit mindestens 300 dpi (Aufloesung).
- Schwarz-weiss oder Farbe, je nach Original.
- Speichern als PDF.

**Bei einem digitalen Dokument** (z.B. E-Mail, Rechnung als PDF):
- E-Mail in Outlook/Thunderbird oeffnen → "Drucken" → "Als PDF speichern".
- Oder die ganze E-Mail als PDF exportieren.

### Schritt 3 — Oben rechts "Anlage K1" einfuegen

Es gibt drei Wege:

**Weg A — In Word/LibreOffice, vor dem Export als PDF:**
- Oben rechts ein Textfeld einfuegen.
- Text: "Anlage K1" (fett, 14pt).
- Position: 2 cm vom oberen Rand, 2 cm vom rechten Rand.
- Datei → Exportieren als PDF.

**Weg B — Im PDF nachtraeglich (mit Adobe Acrobat, Foxit, oder kostenlosen Tools):**
- PDF oeffnen.
- "Stempel"-Funktion oder "Text hinzufuegen".
- Text: "Anlage K1" (fett, 14pt).
- Oben rechts platzieren.
- Speichern.

**Weg C — Mit kostenlosem Online-Tool (z.B. PDF24, ilovepdf):**
- PDF hochladen.
- "Text einfuegen" oder "Annotation".
- Oben rechts "Anlage K1" platzieren.
- Herunterladen.

**Achtung Datenschutz:** Wenn die Anlage sensible Daten enthaelt (Gehaltsabrechnung, Vertrag mit persönlichen Daten), nutzen Sie **keine Online-Tools**, sondern ein lokales Programm.

### Schritt 4 — Seitenzahlen einfuegen (bei mehrseitigen Anlagen)

- Bei mehrseitigen Anlagen: unten zentriert "Seite 1 von 3", "Seite 2 von 3", ...
- In Word: Einfuegen → Seitenzahl → unten zentriert.
- In PDF nachtraeglich: viele PDF-Editoren haben eine "Bates-Nummerierung" oder "Seitenzahlen einfuegen"-Funktion.

### Schritt 5 — OCR aktivieren (optional, aber empfohlen)

Wenn Sie einen gescannten Vertrag haben, ist der Text im PDF nur als Bild gespeichert. Das Gericht kann ihn nicht durchsuchen. **OCR** wandelt das Bild in durchsuchbaren Text um.

- In Adobe Acrobat: Werkzeuge → Text erkennen.
- Kostenlos: Tesseract, OCRmyPDF (Kommandozeile), oder kostenlose Online-OCR-Tools.

Das ist nicht zwingend, aber praktisch für alle.

### Schritt 6 — Dateinamen nach Konvention

Wenn Sie die Anlagen ueber Mein Justizpostfach einreichen, geben Sie den Dateien klare Namen:

```
01_Klageschrift.pdf
02_Anlage_K1_Kaufvertrag.pdf
03_Anlage_K2_Rechnung.pdf
04_Anlage_K3_Email_Beklagter.pdf
05_Anlage_K4_Mahnung.pdf
```

Die fuehrenden Nummern (01, 02, ...) sorgen dafür, dass die Dateien in der **richtigen Reihenfolge** sortiert werden.

## Beispiel — fertige Anlage

So sollte eine fertige Anlage K1 aussehen:

```
+----------------------------------------------+
| Anlage K1 | <-- 14pt fett, oben rechts
| |
| Kaufvertrag | <-- Original-Inhalt
| |
| Zwischen Herrn ABC und Frau XYZ |
| ueber den Verkauf eines Gebrauchtwagens |
| (Marke, Modell, FIN) |
| |
| ... |
| |
| |
| Seite 1 von 3 | <-- unten zentriert
+----------------------------------------------+
```

## Worauf Sie besonders achten muessen

- **Schwarzer Text auf weissem Hintergrund** — kein farbiges Papier, kein Wasserzeichen "ENTWURF" oder "MUSTER".
- **Lesbarkeit**: 12pt ist Standard. Kleiner ist schlecht, groesser unueblich.
- **Datenschutz**: Schwaerzen Sie persönliche Daten Dritter (z.B. Kontonummern fremder Personen), wenn sie nichts mit dem Fall zu tun haben.
- **Dateigroesse**: Mein Justizpostfach hat eine Obergrenze pro Einreichung (typisch 60 MB). Komprimieren Sie grosse PDFs.

## Typische Fehler

- **Fehler:** Anlage K1 ist nur ein verschwommener Handy-Foto-Scan. → **So vermeiden:** Richtig scannen (Scanner-App auf Smartphone oder Multifunktionsdrucker).
- **Fehler:** "Anlage K1" oben links statt oben rechts. → **So vermeiden:** Konvention einhalten — rechts oben.
- **Fehler:** Anlagen ueber 60 MB schicken — beA/MJP weigert sich. → **So vermeiden:** PDF komprimieren (kostenlose Tools).
- **Fehler:** Anlage als Word-Dokument (.docx) statt PDF einreichen. → **So vermeiden:** Immer PDF — als unveraenderbares Format.
- **Fehler:** Anlage K1 enthaelt mehrere Beweise zusammen. → **So vermeiden:** Eine Anlage = ein Beweisstueck. Pro Vertrag, Rechnung, E-Mail eine eigene Anlage.
