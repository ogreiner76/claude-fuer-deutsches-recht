# PDF-Erzeugung für Ihre Klage — von Word bis Scanner-App

## Worum geht es?

Das Gericht erwartet heute fast immer **PDF-Dateien**. Diese Skill zeigt Ihnen, wie Sie aus Word, LibreOffice, gescannten Papieren oder Smartphone-Fotos saubere PDFs machen, die das Gericht und die Gegenseite gut lesen koennen.

## Wann brauchen Sie diese Skill?

- Sie haben Ihre Klage am Computer geschrieben und brauchen ein PDF daraus.
- Sie haben Papier-Dokumente (Vertraege, Briefe) und muessen sie einscannen.
- Sie wollen mehrere PDFs zu einem zusammenfuegen.
- Sie wollen ein PDF kleiner machen, weil das Justizpostfach es ablehnt.

## Fachbegriffe (kurz erklaert)

- **PDF**: Portable Document Format. Ein Format, das auf jedem Geraet gleich aussieht.
- **PDF/A**: Eine spezielle Version von PDF für die Archivierung. Das Gericht akzeptiert diese Variante besonders gerne.
- **OCR**: Texterkennung — wandelt ein Bild-PDF in ein durchsuchbares Text-PDF um.

## Schritt-für-Schritt — PDF aus Word/LibreOffice

### Schritt 1 — Klageschrift fertig schreiben

Schriftart Times New Roman oder Arial, 12pt, Zeilenabstand 1,5. Siehe `anlagen-formatieren-k1-k2-pdf-amtsgericht` für Details.

### Schritt 2 — Als PDF exportieren

**In Word:**
- Datei → Exportieren → PDF/XPS-Dokument erstellen → "Veröffentlichen".
- Dateityp: **PDF/A** (wenn moeglich).

**In LibreOffice:**
- Datei → Exportieren als → Direkt als PDF exportieren.
- Reiter "Allgemein" → Haken bei "PDF/A-1a" oder "PDF/A-2b".

### Schritt 3 — PDF pruefen

Oeffnen Sie das fertige PDF und scrollen Sie es durch. Pruefen Sie:

- Sind alle Seiten da?
- Ist die Formatierung erhalten?
- Sind Tabellen, Fettung, Anfuehrungszeichen korrekt?

Wenn ja, ist das PDF einreichbar.

## Schritt-für-Schritt — Papier scannen

### Schritt 1 — Scanner waehlen

**Wenn Sie einen Drucker mit Scanner haben:** Nutzen Sie ihn. Stellen Sie ein:
- Aufloesung: 300 dpi (Punkt pro Zoll).
- Schwarz-weiss oder Farbe, je nach Original.
- Dateiformat: PDF.

**Wenn Sie nur ein Smartphone haben:** Nutzen Sie eine Scanner-App:
- **Adobe Scan** (kostenlos, gut)
- **Microsoft Lens** (kostenlos)
- **CamScanner** (kostenlos, aber Werbung)
- **Apple Notizen** (auf iPhone, eingebaut)

Diese Apps richten das Dokument automatisch gerade aus und entfernen Schatten.

### Schritt 2 — Sauber fotografieren

- Legen Sie das Dokument auf einen flachen Untergrund mit gutem Licht.
- Vermeiden Sie Schatten und Reflektionen.
- Fotografieren Sie senkrecht von oben.
- Pruefen Sie das Ergebnis: Ist alles lesbar?

### Schritt 3 — OCR (Texterkennung) aktivieren

In der Scanner-App gibt es meistens eine Funktion "Text erkennen" oder "OCR". Aktivieren Sie sie. Dann wird das PDF **durchsuchbar** — das Gericht kann nach Worten suchen.

### Schritt 4 — Als PDF exportieren

In der Scanner-App: "Teilen" oder "Exportieren" → PDF. Auf den Computer uebertragen (per E-Mail, USB, Cloud).

## PDFs zusammenfuegen

Wenn Sie z.B. die Klageschrift und das Anlagenverzeichnis in einer Datei haben wollen:

**Kostenlose Tools:**
- **PDF24** (kostenlos, Windows, Online): Drag-and-Drop, zusammenfuegen.
- **PDFsam Basic** (kostenlos, Windows/Mac/Linux): Open Source.
- **Foxit PDF Reader** (kostenlos, eingeschraenkt).
- **Vorschau** (auf Mac, eingebaut).

**Vorsicht bei Online-Tools:** Wenn Ihr PDF sensible Daten enthaelt (z.B. Personalausweis-Kopie), nutzen Sie **lokale Programme**, nicht Online-Tools. Online-Dienste laden Ihre Datei auf einen fremden Server.

## PDF komprimieren

Mein Justizpostfach hat eine **Obergrenze von rd. 60 MB pro Einreichung**. Wenn Ihr PDF groesser ist, muessen Sie es komprimieren.

**Kostenlose Tools:**
- **PDF24** (kostenlos): "PDF verkleinern".
- **SmallPDF** (online, kostenlos bis zu einer bestimmten Zahl von Dateien pro Tag).
- **PDFsam Enhanced** (kostenpflichtig).

**Vorgang:**
- PDF in das Tool laden.
- Komprimierungs-Stufe waehlen (z.B. "mittel" oder "klein").
- Das Tool gibt ein kleineres PDF aus.
- Pruefen Sie: ist es noch lesbar?

## Dateinamen-Konvention

Geben Sie Ihren PDFs aussagekraeftige Namen:

```
2025-05-27_Klageschrift_Mueller-vs-Schmidt.pdf
2025-05-27_Anlage_K1_Kaufvertrag.pdf
2025-05-27_Anlage_K2_Rechnung.pdf
```

Das Datum am Anfang sorgt für chronologische Sortierung. Der Inhalt im Namen hilft Ihnen selbst, das Richtige zu finden.

**Vermeiden Sie:**
- Leerzeichen ("Anlage K1.pdf" - geht, aber unterstrich ist besser)
- Umlaute im Dateinamen (manche Systeme stolpern: "Anlage_Kuendigung.pdf" statt "Anlage_Kündigung.pdf")
- Sonderzeichen (kein "&", "/", "%", ":")

## Worauf Sie besonders achten muessen

- **PDF/A bevorzugen** für Klageschrift und eigene Schriftsaetze (Archiv-Format).
- **Lesbarkeit pruefen** vor dem Absenden.
- **Datenschutz** bei Online-Tools: lokale Programme nutzen, wenn sensible Daten dabei sind.
- **Dateigroesse**: Wenn ueber 60 MB → komprimieren.
- **OCR**: Bei gescannten Dokumenten immer Texterkennung aktivieren, wenn moeglich.

## Typische Fehler

- **Fehler:** PDF wird per Foto aus dem Schraegwinkel gemacht, Text ist verzerrt. → **So vermeiden:** Scanner-App nutzen (richtet automatisch aus).
- **Fehler:** Word-Datei eingereicht statt PDF. → **So vermeiden:** Vor dem Absenden immer als PDF exportieren.
- **Fehler:** PDF ist 200 MB gross, MJP weigert sich. → **So vermeiden:** Komprimieren oder in mehrere Einreichungen aufteilen.
- **Fehler:** Klageschrift und Anlagen in einem riesigen PDF — Gericht muss alles durchscrollen. → **So vermeiden:** Klageschrift = ein PDF, jede Anlage = ein eigenes PDF.

## Quellen und Aktualitaet

Stand: 05/2026. Mein Justizpostfach (MJP) seit 2024 in Betrieb. PDF/A ist das vom Gericht bevorzugte Format. Praxis-Hinweise zu Tools sind unverbindliche Empfehlungen.
