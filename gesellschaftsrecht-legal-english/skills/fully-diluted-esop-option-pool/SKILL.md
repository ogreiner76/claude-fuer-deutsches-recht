# Fully Diluted, ESOP und Option Pool

## Bedeutung

`Fully diluted basis` ist keine natürliche Rechengröße. Sie ist eine definierte Sicht auf die Beteiligungsverhältnisse, als wären bestimmte zukünftige oder bedingte Rechte schon ausgeübt.

## Intake

1. Welche Instrumente liegen vor: echte Anteile, Optionen, VSOP, Wandeldarlehen, SAFE-ähnliche Rechte, Warrants?
2. Wird der Option Pool pre-money oder post-money geschaffen?
3. Wer trägt die Verwässerung?
4. Gibt es ausgeschlossene Instrumente?
5. Welche Rundung und welcher Stichtag gelten?

## Rechenlogik

| Schritt | Prüfung |
| --- | --- |
| Ausgangskapital | Stammkapital, Geschäftsanteile, laufende Nummern |
| Neue Anteile | Kapitalerhöhung, Investor Shares, Ausgabeaufschlag |
| Pool | ESOP/VSOP pre-money oder post-money |
| Wandlung | Convertible Loan, Discount, Cap, Interest |
| Ergebnis | pre-money, post-money, fully diluted, as-converted |

## Output

- Definition von fully diluted basis.
- Tabelle der einbezogenen Instrumente.
- Verwässerungsmatrix.
- Rückfragen zur Berechnung.
- Warnhinweis, ob eine Register-/Notar-/Satzungsebene betroffen ist.

## Didaktischer Arbeitsmodus

- Erklaere jeden Begriff zweispurig: englischer Praxisbegriff, naheliegende deutsche Entsprechung und warum beides nicht deckungsgleich sein muss.
- Arbeite immer an der Dokumentstelle: Term Sheet, SHA, SPA, Satzung, Gesellschafterliste, Cap Table, Board Paper oder Closing Checklist.
- Gib eine Anfängerfassung in klarer Sprache und eine Praxisfassung fuer die Partnerin oder den Mandanten.
- Wenn ein common-law-Begriff nach deutschem Recht verwendet wird, markiere Auslegungsrisiko, deutsches Ersatzkonzept und Formulierungsvorschlag.

## Ausgabeformat

1. Kurzantwort.
2. Begriff entschluesselt.
3. Deutsche Rechtsfunktion.
4. Wirtschaftlicher Effekt.
5. Typische Fallen.
6. Naechster Drafting- oder Review-Schritt.
