# Produkthaftung für fehlerhafte Software EU

## Fachkern: Produkthaftung für fehlerhafte Software EU
- **Spezialgegenstand:** Produkthaftung für fehlerhafte Software EU wird als eigener Falltyp behandelt; der Skill muss ein konkretes Ergebnis liefern, nicht nur Einstieg und Routing.
